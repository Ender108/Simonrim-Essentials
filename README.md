# Description
Simonrim Essentials is a Wabbajack modlist for those who want a modlist setup built of off Simon Magus' series of gameplay overhauls that they can easily add on to. It can also be used as a standalone modlist for those who a pure Simonrim experience with no extra visual or gameplay features.

# Features
- Mod Organizer 2 installation with Stock Game folder setup﻿
- Essential modding tools (CK Platform Extended, xEdit, Synthesis, xLODGen, DynDOLOD, NGIO, Pandora)
- Essential SKSE plugins and bug fixes
- SkyUI and other essential interface elements
- Full Simonrim suite
- ZERO visual mods or any pre-generated LOD

# Requirements
- Skyrim Special Edition v1.6.1170 (the latest Steam version)
- Skyrim Special Edition Creation Kit v1.6.1378 (the latest Steam version)
- NOTE: This modlist DOES NOT require the paid Anniversary Edition content, nor does it support it.

# Installation
1. Ensure [Visual C++ x64﻿](https://www.techpowerup.com/download/visual-c-redistributable-runtime-package-all-in-one/) is installed. If not, download it from the link, extract the archive, and run install_all.bat as admin. Restart your PC afterwards.
2. Ensure you have a clean, unmodified copy of Skyrim Special Edition and the Skyrim Special Edition Creation Kit installed. It must installed outside of your Program Files folders (e.g. "C:\Steam\steamapps\common\Skyrim Special Edition")
3. Download and install Wabbajack anywhere outside of your Programs Files folders (e.g. "C:\Wabbajack"). Run Wabbajack.exe.
4. Click Settings in the bottom left, then click Log in next to Nexus Mods. Proceed to log in and authorize.
5. Click Browse lists, make sure Non-featured is checked, and then search Simonrim Essentials. Click on the box and then click Download & Install.
6. Set the Installation Location and Download Location anywhere outside of your Programs Files folders and outside your Wabbajack installation folder (e.g. "C:\Modlists\Simonrim Essentials" and "C:\Modlists\Simonrim Essentials\downloads").
7. Click Install in the bottom and wait until it says Finished installing Simonrim Essentials at the top. If any of the files fail to download, you will need to download them manually from their page and place it in the specified download folder.
8. Go to your Simonrim Essentials install folder and launch ModOrganizer.exe, Make sure SKSE is selected in the top left drop-down and click Run if you wish to play.
